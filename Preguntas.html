¿Que es HTML y cual es su funcion principal en el desarrollo web?

HTMl es un lenguaje de enmarcado de hypertexto, en el cual se utilizar para darle forma a un sito web, y su funcion principal es darle arquitectura a una pagina web como las bases de lo que es.

Pregunta 2

¿Que son las etiquetas HTML y como se utilizan?

Las etiquetas HTML son elementos que se caracterizan en dar la semantica y estructura al HTMl, se utilizan para poder poner un texto, listas ordenadas y desordenadas, la semantica en html como lo puede ser el nav, body, footer, header, aside, section, etc.

Pregunta 3

¿Que es css y como se utiliza par dar estilo en una pagina web?

El css es un archivo en el cual se le conecta al HTML, se puede conectar ya sea creando un archivo o de manera interna con una etiqueta style, se utiliza con "id" o "clases" tambien se pueden mandar a llamar las etiquetas, en la cual se puede agregar un color diferente o agregar media querys, y keyframes para poder hacer un diseño dinamico

Pregunta 4

¿Cuales son las propiedades mas importantes del css?

De las propiedades mas importantes de css estan: textAlign, border, background, fontSize, fontFamily, margin, padding, alignItems, flex, displayFlex, @media, etc.

Pregunta 5

¿Que es box model en css?

El modelo de cajas en HTML y css, es la explicacion de que cada elemento tiene una caja imaginaria en la cual tiene un borde, un padding interno, y un margin externo lo cual ordena a los elementos a que no esten sobremontado en otro elemento y genere un error visual se utiliza en el css, y es para dar orden de elementos entre elementos.

Pregunta 6

¿Que es el diseño repsponsivo y como se logra en css?

El diseño responsivo es lo adaptable que puede ser un sitio o pagina web tanto como sea en computadora, televisores, telefonos, tablets, laptops y se logra de una manera un poco compleja en css, ya que normalmente en la mayoria de componentes que esten en un box model se le utiliza un elemento llamado @media en el cual tenemos que poner que al tener dicho dispostivo como puede ser un telefono 360px X 360px sea adapte a esa pantalla.

Pregunta 7

¿Como JavaScript interactua con HTML y CSS atraves del DOM?

El dom es el modelo de objetos en el cual define los elementos como objetos, las propiedades como objetos, el acceso a todos los elementos html, como puede ser los botones y crear eventos o funciones.

Pregunta 8

Ejemplos de interactividad con JS

JavaScript puede hacer que una pagina sea interactiva como por ejemplo de esta manera, existe un boton que en base de datos inserta una producto al carrito, ahora pide una tarjeta de credito que esta validada en la base de datos, al dar click a ese boton la valida y dependiendo de si esta bien o mal, no logra efectuar el pago o si lo logra efectuar, la interactividad se obtiene mediante todo eso, y un mensaje de alerta "un alertDialog" en el cual muestra el problema de transaccion fallida o existosa correctamente.
