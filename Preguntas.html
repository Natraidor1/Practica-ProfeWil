¿Que es HTML y cual es su funcion principal en el desarrollo web?

HTMl es un lenguaje de enmarcado de hypertexto, en el cual se utilizar para darle forma a un sito web, y su funcion principal es darle arquitectura a una pagina web como las bases de lo que es.

¿Que son las etiquetas HTML y como se utilizan?

Las etiquetas HTML son elementos que se caracterizan en dar la semantica y estructura al HTMl, se utilizan para poder poner un texto, listas ordenadas y desordenadas, la semantica en html como lo puede ser el nav, body, footer, header, aside, section, etc.

¿Que es css y como se utiliza par dar estilo en una pagina web?

El css es un archivo en el cual se le conecta al HTML, se puede conectar ya sea creando un archivo o de manera interna con una etiqueta style, se utiliza con "id" o "clases" tambien se pueden mandar a llamar las etiquetas, en la cual se puede agregar un color diferente o agregar media querys, y keyframes para poder hacer un diseño dinamico

¿Cuales son las propiedades mas importantes del css?
